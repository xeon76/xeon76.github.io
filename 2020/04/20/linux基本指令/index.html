linux基本指令

普通用户

文件和目录查看类

  1. ls
    功能:列出当前文件下所有的文件与目录
  2. ls -l
    功能:用列表形式展现当前目录下的文件与目录
  3. ls -lh
    功能：人性化显示
  4. cat
    功能:用于显示文件内容
  5. pwd
    功能:用于显示当前目录

文件和目录管理类

  1. chmod
    功能：用于改变文件权限
    chmod g+w 文件名 功能：组权限中加入写权限
    chmod g-w 文件名 功能：组权限中删除写权限

  2. chgrp
    使用前提:
    这个指令要在root权限下进行，但是ubuntu系统自身是不带root，首先要开启root
    sudo passwd root //会让你输入当前用户密码。输入按下回车输入两次root密码
    su root //提示输入root密码。输入即可
    进入root权限后可以操作
    chgrp root Desktop 将Desktop权限设置为root
    chgrp xeon Desktop 将权限设置为以前的权限
    exit 退出root模式

系统信息查看类

  1. who
    功能:查看当前有哪些用户登录到了系统

  2. date
    功能:显示出当前时间

  3. ps
    功能:当前用户创建的进程

  4. ps -e
    功能:系统所有的进程

  5. kill
    功能:杀死系统内的进程（要进入root权限）

  6. uname
    功能:显示系统名称

  7. uname -a
    功能:显示完整名称

超级用户

常见命令

  1. ifconfig
    功能:用于配置网卡

  2. fdisk
    功能:用于磁盘分区（不建议使用）

  3. modprobe
    功能:用于加载一个模块到内核

  4. lsmod
    功能：内核模块
