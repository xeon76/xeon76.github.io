树莓派python环境隔离

发表于 2020-05-28 分类于 树莓派

更新源

第一步 编辑系统源文件

sudo nano /etc/apt/sources.list

第二步 修改

将初始的源使用#注释掉，添加如下两行清华的镜像源。【注意】这里的树莓派系统是Raspbian-buster系统，在写系统源链接时要注意是buster，网上很多教程都是之前stretch版本，容易出错！
deb http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ buster main contrib non-free rpi
deb-src http://mirrors.tuna.tsinghua.edu.cn/raspbian/raspbian/ buster main contrib non-free rpi

第三步 保存执行如下命令 sudo apt-get update，完成源的更新软件包索引。

#安装leafpad
sudo pip install leafpad

安装pyenv

第一步 安装依赖

sudo apt-get install -y make build-essential libssl-dev zlib1g-dev libbz2-dev libreadline-dev libsqlite3-dev wget curl llvm

第二步 安装

$ curl -L https://raw.githubusercontent.com/pyenv/pyenv-installer/master/bin/pyenv-installer | bash 这一步需要酸酸乳，在电脑上下载好命名abc传到树莓派上
执行 sudo chmod 755 abc 改权限
执行 ./abc 这边千万不要加sudo
会遇到一个warming, 将wanming的三行复制到.profile里面
执行 leafpad .profile
export PATH=”/home/pi/.pyenv/bin:$PATH”
eval “$(pyenv init -)”
eval “$(pyenv virtualenv -init -)”
将这三行复制到文件最下面 保存
执行 source .profile

第三步 安装初始py

pyenv install 3.7.3

第四步 创建环境

pyenv virtualenv 3.7.3 py373

第四步 进入环境

pyenv activate py373
pyenv deactivate py373 (退出环境)

在pyenv环境中更换pip清华源

执行以下指令

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pip -U
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

在环境中安装jupyterlab

第一步

pip3 install jupyterlab

第二步 生成密钥

输入python 进入python环境
执行命令生成秘钥：
In [1]: from notebook.auth import passwd
In [2]: passwd()
Enter password:
Verify password:
Out[2]: 'sha1:3aa19817cfd5:f4d46129439a2de28aaaa03606360add7ae36323'

第三步

将ip设置为，意味允许任何IP访问
c.NotebookApp.ip = ''
这里的密码就是上边我们生成的那一串
c.NotebookApp.password = 'sha1:3aa19817cfd5:f4d46129439a2de28aaaa03606360add7ae36323'
服务器上并没有浏览器可以供Jupyter打开
c.NotebookApp.open_browser = False
监听端口设置为8888或其他自己喜欢的端口
c.NotebookApp.port = 8888
允许远程访问
c.NotebookApp.allow_remote_access = True
保存退出
以上就是配置jupyter过程

树莓派初识

发表于 2020-05-19 更新于 2020-05-20 分类于 树莓派

初识树莓派linux

控制台切换快捷键

图形界面–>字符终端 ctrl+alt+fn
字符终端–>字符终端 alt+fn
字符终端–>图形界面 alt+f7
用户名：pi
密码：raspberry

字符终端的命令提示符

pi@raspberrypi:~ $
用户名:pi
主机名：raspberrypi
当前所在工作目录: ~代表用户所在工作目录
用户类型：$字符代表当前登录的是普通用户，#字符代表的是当前登录的是管理员用户

简单常用的命令

exit : 退出当前命令操作环境
uname -a ：查看内核版本
cat /proc/cpuinfo :查看cpu信息
cat /proc/meminfo :查看内存
hostname : 查看主机名
ifconfig eth0 : 查看IP地址
sudo init 0 :关机
sudo reboot :重启
sudo shutdown -r +分钟 “will rebooted” :定时重启

初识linux命令

Ctrl+U 快速删除光标之前的所有字符
Ctrl+K 快速删除光标之后的所有字符
Ctrl+L 快速清除屏幕字符相当于内部命令“clear”的功能
Ctrl+C 取消当前命令的编辑，并切换为新的一行命令提示符

查看及切换目录

pwd 显示用户当前所在的工作目录位置
使用“cd –”命令切换到上一次执行cd命令前的目录
