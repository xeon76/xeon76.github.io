异常之多个except

类型

情况1：
try:
有可能产生多种异常
except 异常的类型1:
print()
except 异常的类型2
print()
except Exception(): #这个范围很大 所以要放在最后 因为这个这个except是重上往下执行的
pass

如果是多个except 异常类型的顺序要注意，最大的Exception要放在最后

情况2：
try:
有可能产生多种异常
except 异常的类型1:
print()
except 异常的类型2
print()
except Exception() as err:
print(err) —>err的内容就是错误原因

小结

处理异常
try:
可能发生的异常代码
except:
有异常才会进入的代码
else:
没有异常才会进入的代码 如果try中有return 就不会进入else
finally:
无论有没有异常都会执行的代码 如果try中有return,不会对finally造成影响
