树莓派摄像头安装及测试

发表于 2020-05-13 分类于 树莓派

安装

树莓派4B有两个摄像头接口，我们使用距离hdmi近的那个，然后将接口上黑色边框往上拔起，将摄像头的金手指面向hdmi插入，最后按下黑色边框。

设置

首选项 -> Raspberry Pi Configuration -> interfaces -> Camera设置为enable
